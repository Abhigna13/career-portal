# 🎓 AI Career Recommendation Portal
A Django-based AI Career Recommendation Portal that helps students and job seekers find suitable career paths efficiently.

The system includes authentication, a dashboard, and career recommendation features with skill-based job suggestions.

---

## 📌 Project Overview

This project is designed to help users find suitable career paths based on their skills and interests.

It provides a simple interface for users to view career suggestions, skill details, and job recommendations.

The application is built using Django framework.

---

## ✨ Features

✔️ User Login and Registration  
✔️ Career Recommendation System  
✔️ Skill Management  
✔️ Job Listings  
✔️ User Dashboard  
✔️ Admin Panel

## ⚙️ Installation & Setup

1. Clone Repository  
```bash
git clone <repository-link>
```

2. Navigate to Project Folder  
```bash
cd Django_Project_Code/ABHIGNA_14_MAY
```

3. Install Dependencies  
```bash
pip install django
```

4. Run Migrations  
```bash
python manage.py migrate
```

5. Run Server  
```bash
python manage.py runserver 8001
```

6. Open Browser  
```
http://127.0.0.1:8001/
```
